## Code Blocks

### Inline Code

Use the `bun dev` command to start the development server. Configure options in `config.json`.

### Code Block (No Language)

```
Plain text code block without syntax highlighting.
Useful for output or generic content.
```

### Code Block (Bash)

```bash
# Install dependencies
bun install

# Start development server
bun dev

# Build for production
bun run build
```

### Code Block (TypeScript)

```typescript
interface Config {
  port: number;
  host: string;
  debug: boolean;
}

const config: Config = {
  port: 3000,
  host: 'localhost',
  debug: true,
};

export function getConfig(): Config {
  return config;
}
```

### Code Block (JSON)

```json
{
  "name": "promptty",
  "version": "1.0.0",
  "scripts": {
    "dev": "bun run src/index.ts",
    "build": "bun build src/index.ts"
  }
}
```

### Code with Title and Line Highlighting
